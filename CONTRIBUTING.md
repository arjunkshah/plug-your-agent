# Contributing

Thanks for your interest in Agent Plugin Bar.

## Development setup

1. Install dependencies: `npm install`
2. Run the docs/demo site: `npm run dev`
3. Build packages: `npm run build`

## Pull requests

- Keep changes focused and scoped.
- Update or add documentation in `README.md` when behavior changes.
- Avoid introducing new dependencies unless necessary.
- Run `npm run build` before submitting.

## Issues

Please include:
- reproduction steps
- expected vs actual behavior
- environment details
