# agentbar-cli

Global wrapper for the Agent Plugin Bar CLI.

## Install

```bash
npm install -g agentbar-cli
```

## Usage

```bash
agentbar init
agentbar snippet
agentbar set siteUrl https://your-site.com
agentbar stats
```

`agentbar init` only asks for your site URL. This package forwards to
`@arjun-shah/agentbar-cli`.
